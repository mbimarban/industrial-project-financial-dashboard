# Financial Control Dashboard – Industrial Projects

## 🔷 Overview

This project presents an executive-level financial dashboard designed for industrial project portfolio management.

The dashboard is built to support decision-making at management level by providing clear visibility on:

- cost overruns  
- margin erosion  
- cash collection risk  
- overall portfolio performance  

It is not just a reporting tool — it is designed as a **decision-support system**.

👉 [Open Financial Control Dashboard](https://mbimarban.github.io/industrial-project-financial-dashboard/)

Open the live dashboard to explore project margin, cost overruns and cash flow risks in an interactive format.

---

## 🔷 Key Features

- **WHAT / SO WHAT / NOW WHAT logic**  
  The dashboard shows:
  - what is happening  
  - what it means  
  - what actions should be taken  

- **Executive Summary (decision panel)**  
  Immediate visibility of:
  - portfolio margin  
  - cost overruns  
  - cash gap  
  - collection rate  

- **Dynamic Recommended Actions**  
  Automatically identifies:
  - top overrun projects  
  - projects with highest cash gap  

- **Scenario Analysis (Base / Risk / Recovery)**  
  Simulates impact on margin:
  - cost increase (risk case)  
  - recovery actions  
  - expected improvement  

- **Project-level diagnostics**  
  Detailed view of:
  - contract vs forecast (EAC)  
  - margin per project  
  - invoicing vs cash

## 🔷 Purpose

The goal of this project was to demonstrate how financial data can be transformed into a **decision-oriented dashboard**, not just a static report.

It supports management in:

- identifying financial risks early  
- prioritizing actions  
- understanding impact of different scenarios  
- improving control over project performance
